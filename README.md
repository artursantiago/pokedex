<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/artursantiago/pokedex.git
   ```
2. Install the packages
   ```sh
   yarn
   ```
3. Run the local development server
   ```sh
   yarn start
   ```
